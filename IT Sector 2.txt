Information Technology (IT) & Software

1) Web Development (Full Stack)
A complete path from fundamentals (HTML/CSS/JS) to modern front-end frameworks, back-end APIs, databases, authentication, testing, DevOps, and deployment.
Prerequisites
•	Basic computer literacy and internet usage
•	Optional: Prior programming basics
Tools/Stack
•	Frontend: HTML5, CSS3, JavaScript (ES6+), TypeScript (optional), React/Vue/Angular
•	Backend: Node.js, Express (or NestJS)
•	Databases: PostgreSQL/MySQL, MongoDB
•	DevOps: Git/GitHub, Docker, CI/CD (GitHub Actions), Nginx
•	Testing: Jest, React Testing Library, Cypress/Postman
•	Build: Vite/Webpack, Babel, ESLint/Prettier
Detailed Modules
1.	Web Foundations
o	How the web works (HTTP/HTTPS, DNS, request/response)
o	HTML5 semantics, forms, accessibility (a11y)
o	CSS3 fundamentals, Box Model, Flexbox, Grid, responsive design
o	Developer tools, browser rendering, performance basics
2.	Modern JavaScript (ES6+)
o	Variables, scope, closures, this, prototypes
o	Modules, async/await, promises, fetch API
o	Data structures, array methods, error handling
o	TypeScript intro (types, interfaces, generics)
3.	Front-End Framework (React example)
o	Components, JSX, props/state, hooks
o	Routing, context API, state management (Redux/RTK/Zustand)
o	Forms, validation, controlled vs uncontrolled
o	Performance optimization, suspense, code splitting
4.	UI/UX & Styling Systems
o	Design systems, BEM, CSS Modules, Tailwind/Styled Components
o	Accessibility best practices (WCAG), ARIA
o	Theming, dark mode, internationalization basics
5.	Backend with Node.js & Express
o	REST API design, controllers, services, routers
o	Middleware, error handling, rate limiting
o	Authentication/Authorization (JWT, sessions, OAuth2)
o	Security: Helmet, CORS, input validation
6.	Databases & ORM/ODM
o	SQL design, normalization, indexing, transactions
o	PostgreSQL & MySQL basics; joins and query optimization
o	NoSQL with MongoDB, schema design
o	ORM tools (Prisma/Sequelize/Mongoose)
7.	Testing & Quality
o	Unit/integration testing with Jest and Supertest
o	Frontend tests with React Testing Library, Cypress (E2E)
o	API testing with Postman/Newman
o	Linting, formatting, code review practices
8.	DevOps Essentials
o	Git workflows (Feature branches, PRs)
o	CI/CD pipelines (build, test, deploy)
o	Dockerization, multi-stage builds, docker-compose
o	Nginx reverse proxy, SSL/TLS basics
9.	Deployment & Cloud
o	Deploy to VMs (Ubuntu), PaaS (Render/Heroku), or cloud (AWS/Azure)
o	Environment variables, secrets management
o	Monitoring, logging (Winston), error tracking (Sentry)
o	Basic scalability: horizontal vs vertical scaling, caching (Redis)
10.	Performance & Advanced Topics
•	Web performance metrics (LCP, TTI), Lighthouse
•	Caching strategies (HTTP cache, CDN)
•	GraphQL basics (optional)
•	WebSockets/Socket.IO for real-time features
Labs & Projects
•	Build a responsive portfolio site
•	RESTful API with authentication + CRUD + Postman tests
•	Full-stack project: E-commerce or social app with admin dashboard
•	CI/CD pipeline + containerized deployment
Capstone
•	Production-ready full-stack app (secure auth, payments, role-based access, tests, docs, deployed)

2) Prompt Engineering
Overview
Designing, evaluating, and operationalizing prompts for LLMs to achieve reliable, safe, and efficient outputs across tasks (content, coding, data extraction, agents).
Prerequisites
•	Basic Python and API usage
•	Understanding of NLP/LLMs helpful but not required
Tools
•	OpenAI/other LLM APIs, LangChain/LlamaIndex (optional)
•	Python, Jupyter/VS Code
•	Evaluation: BLEU/ROUGE-style metrics (contextual), human eval sheets
•	Vector DBs (FAISS/Chroma), basic retrieval
Detailed Modules
1.	LLM Fundamentals
o	How LLMs work conceptually (tokens, context windows)
o	Zero-shot, few-shot, CoT (chain-of-thought overview), self-consistency
o	Tokenization, temperature/top-p, system vs user prompts
2.	Prompt Patterns
o	Instruction clarity, role prompting, constraints, formats
o	Decomposition, step-by-step plans, checklists
o	Examples for classification, extraction, transformation, generation
3.	Structured Outputs
o	JSON schemas, robust formatting
o	Output validation, retry strategies
o	Guarding against hallucinations (ask for sources, verification steps)
4.	Retrieval-Augmented Generation (RAG)
o	Chunking, embeddings, similarity search
o	Context construction, citations, freshness
o	Handling long documents and multi-turn memory
5.	Tool Use and Agents (Intro)
o	Function calling / tool APIs
o	Planning & execution loops, stopping criteria
o	Safety, authorization, rate limits
6.	Evaluation & Benchmarks
o	Defining success criteria, test suites
o	Human-in-the-loop evaluation, rubrics
o	A/B testing prompts, regression testing
7.	Safety, Bias, and Governance
o	Red-teaming prompts, jailbreak resistance
o	Bias mitigation, tone control, compliance considerations
o	Content filters and policies integration
8.	Cost, Latency, and Scaling
o	Model selection, context budgeting
o	Prompt compression, caching responses
o	Observability and logging
9.	Domain-Specific Prompting
o	Code generation and debugging
o	Data extraction and summarization
o	Creative writing and marketing
10.	Productionization
•	Versioning prompts
•	Canary deployments of new prompts
•	Documentation and collaboration practices
Labs & Projects
•	Build a robust prompt suite for a Q&A chatbot with JSON outputs
•	Implement a simple RAG pipeline over custom documents
•	Create an evaluation harness with test cases and scoring
Capstone
•	Production-grade prompt system for a chosen domain (e.g., support bot) with evals, safety, and cost controls

7.	Unsupervised Learning

o	Customer segmentation
8.	Time Series

o	Prophet basics
o	Forecast evaluation and seasonality
9.	SQL & Data Warehousing
o	Advanced SQL joins, window functions, CTEs

10.	Communication & BI
•	Dashboarding in Power BI/Tableau

11.	MLOps (Intro)
•	Model tracking (MLflow), reproducibility
•	Basics of deployment options
•	Monitoring drift and model performance
Labs & Projects
•	EDA and modeling on public datasets (Kaggle-style)
•	A/B test analysis with recommendations
•	Build a dashboard for stakeholder insights
Capstone
•	End-to-end ML solution: data ingestion, EDA, model, evaluation, dashboard, business recommendation



4) Cyber Security & Ethical Hacking
Note: This course emphasizes legal, ethical, and authorized testing only. All offensive techniques are taught strictly for defense and within isolated lab environments.
Prerequisites
•	Networking basics (TCP/IP), Linux command line
•	Basic scripting (Python/Bash) helpful
Tools
•	Kali Linux/Parrot OS in VM, Wireshark, Nmap
•	Burp Suite Community, OWASP ZAP
•	Metasploit (lab-only), Hashcat/John (lab-only)
•	SIEM basics (Splunk/ELK), OSINT tools (theHarvester)
•	OWASP Top 10 resources
Detailed Modules
1.	Security Fundamentals
o	CIA triad, risk, threat modeling, attack surfaces
o	Legal frameworks, scope of work, rules of engagement, reporting
2.	Networking & System Security
o	TCP/IP, routing, DNS, DHCP
o	Linux/Windows hardening, firewall basics
o	Logging and monitoring principles
3.	Reconnaissance & Enumeration (Defensive Perspective)
o	OSINT methods, metadata analysis
o	Network scanning (Nmap), service enumeration (legal lab use only)
o	Interpreting findings and mitigations
4.	Web Application Security (OWASP Top 10)
o	Injection, XSS, broken auth, sensitive data exposure
o	Insecure design, SSRF, deserialization
o	Secure coding practices, secure headers
5.	Vulnerability Assessment
o	Automated scanning vs manual verification
o	False positives, prioritization (CVSS)
o	Patch management and remediation plans
6.	Exploitation Basics (Lab-Only)
o	Safe exploitation scenarios in controlled labs
o	Metasploit concepts, payloads (for understanding, not real-world use)
o	Defense strategies post-discovery
7.	Wireless & Network Attacks (Lab-Only)
o	Wi-Fi security (WPA2/WPA3)
o	Sniffing, MITM concepts in lab environment
o	Network segmentation and defense
8.	Cryptography & PKI
o	Symmetric/asymmetric crypto, hashing, TLS/SSL
o	Key management, certificates, HSM basics
9.	Incident Response & Forensics
o	Incident lifecycle, triage, containment, eradication, recovery
o	Disk/memory forensics basics, chain of custody
o	Reporting and communication
10.	Security Operations & Governance
•	SIEM use cases, detections
•	Security policies, awareness training
•	Compliance (ISO 27001, SOC 2, PCI-DSS basics)
Labs & Projects
•	Web app vulnerability assessment in DVWA/Juice Shop (lab)
•	Network scan and secure configuration plan
•	Incident response tabletop exercise with report
Capstone
•	Full penetration test simulation in a lab with executive report and remediation roadmap
